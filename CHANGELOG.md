# 更新日志

## v0.4.2 - 2026-07-19

- 修复后端安装器把旧版本备份留在 `SillyTavern/plugins/` 下，导致酒馆把 `theater-favorites.backup-*` 误当成多个服务器插件的问题。
- 旧备份会在重新安装时自动迁移到 `SillyTavern/backups/theater-favorites/`；今后的后端和 `config.yaml` 备份也统一保存在这里。
- 为复制式安装的收藏夹后端增加本地 Git 边界，阻止服务器插件自动更新器向上误认 SillyTavern 根仓库，从而避免收藏夹触发 `detected dubious ownership`。
- 安装过程改为先写入临时目录、成功后再替换正式后端；失败时会尝试恢复旧后端，降低安装中断造成插件缺失的风险。
- Windows 批处理入口现在会保留 Node 安装器的失败状态；安装失败时不再继续显示误导性的 `Done`。
- 不自动修改 Git 全局 `safe.directory`，也不改动其他服务器插件。若 tavern-notes 等其他插件仍报告相同错误，需要单独处理酒馆目录所有权或关闭服务器插件自动更新。

## v0.4.1 - 2026-07-19

- 修复 Windows 后端安装器 `install-server-plugin.bat` 使用 LF 换行时，`cmd.exe` 会把带空格的安装路径拆坏，导致出现“不是内部或外部命令”的问题。
- 修复后端安装器提示文本在部分 Windows 控制台中乱码的问题；批处理入口改为纯英文提示，实际安装器继续输出中文结果。
- 修复 `install-server-plugin.js` 中安装提示变量路径不够完整的问题，失败时可手动传入 SillyTavern 根目录。
- 新增 Markdown 渲染快照识别：标签型小剧场原文像 Markdown、且酒馆消息里已经渲染出粗体、斜体、引用、代码、链接或列表时，收藏夹会保存酒馆渲染后的安全快照用于预览；拿不到快照时仍按纯文本保底。
- 为 Markdown 快照增加收藏夹内的稳定预览样式，避免离开聊天区后因为主题样式继承不完整，出现引用块、标题或段落排版和原消息明显不一致。
- Markdown 标签预览现在会优先从原文安全渲染标题、引用块、段落换行和基础行内样式；已收藏为纯文本标签的旧条目也可恢复 Markdown 引用/对话预览。
- 修复展开长收藏后再删除条目时，浏览器的 `scrollIntoView()` 会连带滚动固定弹窗外壳，导致标题、关闭、刷新和设置按钮被裁出顶部的问题；现在只滚动收藏列表，并持续锁定弹窗外壳滚动位置为零。
- Markdown 阅读预览会隐藏 `<!-- ... -->` 注释块，不再把系统日志、构思校验等注释内容显示在正文中；收藏原文仍会完整保留。
- 新增全局标签管理：可查看每个标签用于多少条收藏，并一次性从全部收藏中移除指定标签，不会删除收藏本身。
- 重做手机端空间分配：标题、收藏数量和顶部操作压缩为紧凑两行；角色、聊天、来源和标签筛选默认收进带状态数量的“筛选”按钮，搜索框保持直接可用。
- 手机端把单条收藏的移动、删除、重命名、编辑和标签操作收进明确的“管理这条收藏”折叠区，并减少列表左右留白，让预览更早出现且获得更大宽度。
- HTML 小剧场在手机端的预览最小高度改为按屏幕高度计算，不再固定为 260px，避免内容和交互按钮挤在过小窗口内。
- 本次不改变收藏数据格式，不需要迁移收藏；由于新增全局标签接口，更新后需要重新运行一次后端安装器并重启 SillyTavern。

## v0.4.0 - 2026-07-18

- 新增编辑正文功能：展开收藏后可以直接编辑小剧场原文并保存，保存后会更新渲染预览、搜索内容、导出内容和去重签名。
- 新增拖动排序功能：列表左侧不再显示序号，改为拖拽把手；拖动收藏即可调整顺序，刷新后顺序会保留。“上移 / 下移”按钮保留为备用操作。
- 旧收藏无需迁移；没有手动排序字段的旧数据会继续按原收藏时间显示，首次调整顺序时自动兼容。
- 强化快捷入口防重复处理，旧入口或主题重建出的重复入口会被隐藏而不是删除，避免快捷回复栏出现两个收藏夹图标或反复闪烁。
- 手机扩展菜单入口改为主题色 mask 图标，不再使用可能显示成破图的小图片。
- 修复纯文字小剧场里嵌套其他识别标签时被误判成 HTML 的问题；这类内容会按纯文本保留换行，不再把文字挤成一段。

## v0.3.1 - 2026-07-18

- 修复日间美化主题下，固定深色收藏夹面板里的筛选框、列表文字和操作按钮颜色被主题覆盖导致看不清的问题。
- 修复部分 HTML 小剧场渲染预览在深色预览画布中出现黑字的问题，同时保留作者明确设置的背景和文字颜色。
- 修复发送栏快捷入口被美化主题拉伸成大型悬浮框、停在输入栏附近且不易关闭的问题；关闭态会强制隐藏，也支持按 `Esc` 关闭。
- 修复纯文字小剧场在渲染预览里丢失换行、文字全部挤在一起的问题；已有收藏无需重新保存。
- 修复手机端快捷回复栏重建后收藏夹入口消失的问题；入口会固定在快捷回复区第一位并自动恢复。
- 快捷入口恢复使用收藏夹原始面具图形，并让按钮外观和图标颜色跟随酒馆当前主题，避免和同排按钮、面板内图标、拟界文库入口画风割裂。
- 热修快捷入口在部分深色主题中被原生按钮样式染成异常底色的问题，入口会继续使用酒馆快捷回复按钮的容器风格。
- 版本号更新到 `0.3.1`。
