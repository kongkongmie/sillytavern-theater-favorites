# 小剧场收藏夹 v0.4.1

一个给 SillyTavern 使用的小剧场收藏、阅读和管理插件。

可以收藏 `<snow>`、HTML/正则渲染、details 折叠小剧场，也可实验性兼容拟界文库。收藏完整保存在自己的 SillyTavern 本地，不会上传到外部服务器。

## 功能

- 聊天中的每个小剧场都有独立收藏按钮
- 多个小剧场可以分别收藏
- 收藏夹一次只展开一条，避免同时渲染太多内容
- 支持搜索、角色、聊天、来源和标签筛选
- 支持重命名、标签、删除、编辑正文和手动排序
- 支持 JSON 完整备份与导入
- 支持导出浏览器可直接打开的 HTML 阅读版
- 使用完整内容 SHA-256 避免重复收藏

## v0.4.1 热修与预览优化

- 修复 Windows 用户双击 `install-server-plugin.bat` 后，出现“不是内部或外部命令”的问题。
- 修复后端安装器提示乱码、路径打印异常的问题。
- 新增 Markdown 渲染快照识别：如果小剧场原文是 Markdown，且酒馆消息里已经渲染出粗体、斜体、引用、代码、链接或列表，收藏夹会保存酒馆渲染后的预览快照；抓不到快照时仍按纯文本显示，避免格式被误处理。
- 修复展开长收藏后删除条目时，收藏夹顶部标题、关闭、刷新和设置按钮可能被裁掉的问题。
- Markdown 阅读预览会隐藏 `<!-- ... -->` 注释块，但不会改动收藏原文。
- 新增全局标签管理，可查看标签使用数量，并一次性从全部收藏中移除不再需要的标签。
- 优化手机端阅读空间：顶部工具栏压缩为两行，四项筛选默认折叠，单条管理操作收进“管理这条收藏”；预览区域更宽，HTML 预览最小高度会跟随屏幕调整。
- 不需要迁移已有收藏；由于标签管理新增了后端接口，更新到 v0.4.1 后请重新运行一次后端安装器并重启 SillyTavern。若 v0.4.0 安装器曾报错，也会一并修复。

## v0.4.0 新功能

- 新增“编辑正文”：展开收藏后可以直接修改小剧场原文，保存后会更新预览、搜索、导出和去重签名。
- 新增拖动排序：列表左侧改为拖拽把手，拖动收藏即可调整顺序，刷新后仍会保留；“上移 / 下移”按钮保留为备用操作。
- 旧收藏无需迁移，更新后可以直接使用。
- 修复快捷回复栏可能出现两个收藏夹图标或重复入口反复闪烁的问题；手机扩展菜单入口不再使用可能破图的小图片。
- 修复纯文字小剧场里嵌套其他识别标签时被误判成 HTML，导致预览文字挤成一段的问题。

## v0.3.1 修复

- 修复使用日间美化时，固定深色收藏夹中的筛选文字、列表文字和按钮变黑看不清的问题。
- 修复部分小剧场预览被移除原背景后出现黑字贴深色底的问题；作者明确设置的配色仍会保留。
- 修复部分美化主题把发送栏快捷入口拉伸成无法关闭的大型悬浮框；关闭后的收藏夹会强制隐藏，也可按 `Esc` 关闭。
- 修复纯文字小剧场在渲染预览中丢失换行、全部挤在一起的问题；已有收藏会直接恢复正常。
- 修复手机端快捷回复栏重建后收藏夹入口消失的问题；入口会固定在快捷回复区第一位并自动恢复，不会挤进输入区。
- 快捷入口恢复使用收藏夹自己的原始面具图形，同时让按钮外观和图标颜色跟随酒馆当前主题；入口仍会锁定安全尺寸与定位，不再出现画风突兀、深色主题底色异常或被主题拉伸的问题。

## 安装

### 1. 在酒馆里安装

打开 SillyTavern 的“扩展 → 安装扩展”，粘贴：

```text
https://github.com/kongkongmie/sillytavern-theater-favorites
```

### 2. 运行一次后端安装器

Windows：双击扩展文件夹里的 `install-server-plugin.bat`。

安卓 Termux / Linux / Mac / 云服务器：

```sh
node SillyTavern/public/scripts/extensions/third-party/theater-favorites/install-server-plugin.js
```

### 3. 重启酒馆

看到“安装完成”后，重启 SillyTavern，再刷新浏览器。收藏夹设置中显示“前端已加载、后端已连接”就是成功。

完整图文安装手册见附件。

## 提醒

- 插件由前端和后端两部分组成，首次安装需要运行一次安装器。
- 平常更新只需使用酒馆扩展更新；只有更新说明特别要求时才重新安装后端。
- HTML 小剧场可能包含作者写入的脚本或外部资源，请只收藏和运行可信内容。
