# 🎭 小剧场收藏夹 v0.4.5

**强大的 SillyTavern 小剧场收藏、阅读与管理插件。**

遇到喜欢的小剧场，不必再翻聊天记录，也不用复制到零散文档里。点一下收藏，就能把它完整保存到自己的 SillyTavern 中，随时搜索、整理、阅读和备份，把喜欢的故事长期珍藏下来。

## 支持哪些小剧场？

- MOM 系 `<snow>` 小剧场：果实、月读、象牙塔、蛾摩拉等均为此格式
- HTML / 正则渲染的小剧场
- Markdown 与纯文字小剧场
- `details` 折叠式文字小剧场
- 实验性兼容拟界文库小剧场

收藏内容完整保存在你自己的 SillyTavern 本地，不会上传到外部服务器。

## ✨ 主要功能

- 聊天中的每个小剧场都有独立收藏按钮，一条回复里的多个小剧场也能分别收藏
- 收藏夹一次只展开一条，避免多个复杂小剧场同时运行导致卡顿
- 支持按标题、角色、聊天、来源和标签搜索筛选
- 支持重命名、编辑正文、删除和拖动排序
- 支持全局标签管理，可查看标签使用数量并一键从全部收藏中移除标签
- 支持 Markdown 标题、引用、对话、换行和基础文字样式预览
- 支持完整 JSON 备份、恢复与跨设备迁移
- 支持导出浏览器可以直接打开的 HTML 阅读版
- 使用完整内容 SHA-256 去重，避免同一小剧场被重复收藏
- 手机端提供紧凑阅读布局，把更多屏幕空间留给小剧场预览

## 🩹 v0.4.5 CSS 显示正则热修

- 修复部分依赖 SillyTavern 页面 CSS 的小剧场，在收藏后只剩结构、样式没有生效的问题
- 自带完整样式或脚本的小剧场继续使用隔离预览；依赖酒馆样式的内容会正常继承对应 CSS

- 从 v0.4.4 更新后刷新页面即可，不需要重新安装后端

## v0.4.4 性能热修

- 修复更新后酒馆持续卡顿：收藏夹不再因为自己添加按钮、菜单和图标而反复扫描全部聊天
- 新消息和流式输出只处理发生变化的消息；点击收藏时再读取最新完整内容
- 智绘姬图片复用改为一次建立索引，不再按每个提示词重复搜索页面和 iframe
- 拟界文库取消固定 1.5 秒轮询，HTML 预览也会忽略无关的样式动画变化
- 新增“更新”页面：每次页面会话首次打开收藏夹时只检查一次版本号；有更新时按钮会出现小圆点和轻微高亮，引导你查看内容并自行决定是否安装，不会定时轮询或自动更新
- 深色收藏夹的文字和按钮颜色固定，不再被日间主题或美化覆盖成看不见的颜色
- 所有入口统一使用 SillyTavern 自带的剧场图标，不再出现图片图标加载失败、彩色破图或不同设备显示不一致
- 修复收藏超过 20 条后翻页栏消失、显示正则与 Markdown 没有应用到收藏预览的问题
- 超长 HTML / 拟界文库预览改为限制在当前屏幕内并在预览内部滚动，不再被巨大空白撑到看不见内容
- 修复删除收藏后原消息仍显示“已收藏”、无法重新收藏的问题
- 修复依赖完整 `<snow>` 标签的显示正则无法触发，以及部分 Markdown 标题、引用之间出现巨大空白的问题
- 修复固定宽度小剧场在手机端右侧被裁切；过宽内容会自动缩放到可视区域
- 标签、更新、设置改为互斥页面，重复点击当前按钮即可回到收藏列表
- 标签管理改为紧凑列表，可批量移除标签；给单条收藏添加标签后可直接连续输入
- 从 v0.4.3 更新后完整刷新页面即可，不需要重新安装后端

## v0.4.3 手机端与入口热修

- 修复部分手机浏览器打开收藏夹后，界面被压成顶部一条黑线的问题
- 修复电脑端与手机端入口图标受主题或浏览器遮罩兼容性影响而消失的问题
- 图标改为直接加载原始图案，并在加载失败时自动显示备用票券图标
- 修复 `<snow>` 内嵌 `<details>` 时折叠标题和正文不渲染的问题；图片生成提示词不会混进阅读正文
- 具有真实网络地址或酒馆相对地址的图片可随正文显示，只保存地址引用；Base64、临时图片和无地址生成图不会复制进收藏
- 安装智绘姬 v2.7.8 时，无地址图片提示词可手动调用智绘姬生成；不会自动生图，按钮会提示可能消耗额度，生成结果也不会写入收藏
- 当前聊天中已有对应智绘姬图片时会直接临时复用，无需再次生成；Base64 图片仍不会写进收藏文件
- 从 v0.4.2 更新无需重新安装后端，也不会影响已有收藏；更新扩展后刷新页面即可

## v0.4.2 安装器热修

- 修复旧后端备份被酒馆误当成多个服务器插件的问题
- 重新运行安装器会自动把 `plugins/theater-favorites.backup-*` 迁移到 `backups/theater-favorites/`

- 不自动修改全局 Git 安全设置，也不改动其他服务器插件

## v0.4.1 更新内容

### 阅读与渲染

- 新增 Markdown 小剧场识别与安全预览，支持标题、引用/对话、段落换行、粗体、斜体、删除线和行内代码

### 收藏管理

- 新增“编辑正文”，可直接修改收藏内容；保存后同步更新预览、搜索、导出和去重签名

- 新增全局标签管理，可统计每个标签用于多少条收藏，并一次性从全部收藏中移除指定标签
- 打开收藏夹时默认回到列表顶部，不再停留在上一次的滚动位置

### 手机与界面

- 单条收藏的移动、删除、重命名、编辑和标签操作收进“管理这条收藏”，让预览更早出现
- 减少手机端左右留白，并让 HTML 预览最低高度跟随屏幕调整
- 修复删除或收起长条目后，关闭、刷新和设置等顶部按钮被裁掉的问题
- 修复快捷入口重复、闪烁、消失、图标破损或被主题拉伸的问题
- 修复日间美化主题下，固定深色收藏夹出现黑色文字和黑色按钮的问题

### 安装器

- 修复 Windows 双击 `install-server-plugin.bat` 时出现“不是内部或外部命令”的问题
- 修复带空格路径解析错误、控制台提示乱码和安装路径显示不完整的问题

## 📦 安装方法

### 1. 在酒馆里安装扩展

打开 SillyTavern 的“扩展 → 安装扩展”，粘贴：

```text
https://github.com/kongkongmie/sillytavern-theater-favorites
```

### 2. 安装后端服务插件

Windows / PC：双击扩展文件夹里的：

```text
install-server-plugin.bat
```

安卓 Termux / Linux / Mac / 云服务器：

```sh
node SillyTavern/public/scripts/extensions/third-party/theater-favorites/install-server-plugin.js
```

### 3. 重启酒馆

安装完成后重启 SillyTavern，再刷新浏览器。收藏夹设置中显示“前端已加载、后端已连接”即安装成功。

## 🔄 从旧版本更新

1. 在 SillyTavern 扩展面板更新“小剧场收藏夹”
2. 重新运行一次后端安装器
3. 重启 SillyTavern
4. 刷新浏览器页面

v0.4.2 不改变收藏数据格式，已有收藏不需要迁移。

## 💾 收藏保存在哪里？

收藏保存在当前 SillyTavern 用户目录：

```text
SillyTavern/data/<当前用户>/theater-favorites/
```

删除扩展不会自动删除收藏文件。仍建议定期使用收藏夹里的“导出备份”，把珍贵的小剧场再保存一份。

> HTML 小剧场可能包含作者编写的脚本或外部资源，请只收藏和运行可信来源的内容。

项目地址：<https://github.com/kongkongmie/sillytavern-theater-favorites>

- 自动隐藏 `<!-- ... -->` 注释内容，不再把系统日志或构思校验显示进正文
- 新增编辑正文、拖动排序和全局标签管理
- 修复纯文字嵌套标签后文字挤在一起的问题
- 重做手机端布局，压缩顶部与管理按钮，把更多空间留给预览
